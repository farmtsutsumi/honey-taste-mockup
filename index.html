📌 このモックでカバーしている MVP 機能

  • 1-A / 1-B · FeliCa打刻（共用端末）→ 完了時に当月の出勤実績と給与見込みを即時フィードバック
  • 2-A · オーナーの多店舗ダッシュボード。店舗別人件費、出勤中人数、前借り残高を一覧
  • 2-B · 前借り申請の承認フロー（オーナーのみ）。スタッフ側申請画面は v2 で追加
  • 2-C · 月末締めで全店全スタッフの支給額を一覧 → 振込用CSV出力
  • 3-A · 店長の勤怠帳簿（今日の出勤状況＋月次累計）
  • 3-B · 週次シフト編集 → 確定でLINE通知
  • 3-C · 個別給与明細。バック・源泉は欄を用意して v2 で自動化

🔮 プレゼン後に詰める論点（合意済）

  • バック種類と料率（テンプレ＋自由項目）／業態別ルール
  • 売上入力フロー — 現状は手書き伝票、デジタル化の運用設計が必要
  • 前借り上限・頻度／店長代理申請の可否
  • シフト希望提出 → 確定フロー、ヘルプ要請・交換
  • スタッフ側スマホアプリ（給与/前借り残高の本人ビュー、LINE通知）
  • 源泉徴収（10.21%/5万円控除）自動計算
  • 振込API直結（GMOあおぞら / freee / 全銀CSV）
  • 本人確認書類アップロード（風営法名簿）
  • 客管理・ボトルキープ・売上ダッシュボード等の拡張機能（店舗ごとON/OFF）
